# Phase 8: AS5311 外置线性编码器 — 计划

## 目标

将电机内置编码器 (半闭环) 替换为 AS5311 线性磁编码器 (全闭环)，直接测量输出端位移，消除齿轮间隙误差，提升定位精度至 ~10µm 级别。

## 改动范围

1. **硬件层**: TIM8 (PC6/PC7) → TIM4 (PB6/PB7)，PA9 → PB8 INDEX
2. **位置参数**: 重新标定所有运动常量 (DEADZONE, HOME_OFFSET 等)
3. **运动控制**: 删除 backlash 两阶段补偿，改为全闭环单向逼近
4. **INDEX 校准**: 利用 INDEX 脉冲做运行时漂移检测
5. **测试**: 适配所有单元测试

## 设计要点

详见 `spec.md`

## 风险

- TIM4 滤波器参数需板上调优 (AS5311 输出信号质量)
- DEADZONE=10 (20µm) 可能过于激进，需实测后调整
- 磁条安装精度影响 INDEX 校准基准
