# ZLENS_DC 产品使用指南

## 产品简介

ZLENS_DC 是电动变倍镜头控制器，通过 UART 串口接收命令，驱动 DC 有刷电机实现 0.6×~7.0× 连续变倍。支持 14 级倍率切换、速度调节、归零校准。

## 接线

| 线缆 | 控制器端 | 主机端 | 说明 |
|------|----------|--------|------|
| TX | PA2 | 主机 RX | 控制器发送 |
| RX | PA3 | 主机 TX | 控制器接收 |
| GND | GND | GND | 共地 |

- 电平：TTL 3.3V
- 串口参数：115200 baud, 8N1, 无流控

供电要求：8.3V~24V DC（低于 8.3V 触发欠压保护）。

## 上电流程

### 首次上电（或 FRAM 数据无效）

```
上电 → 自动归零（电机反向运动到机械限位 → 正向回退到原点）→ 就绪
```

归零约需 5~10 秒。归零完成后设备发送 HOMING_DONE 响应。

### 正常上电（FRAM 有效）

```
上电 → 恢复上次位置 → 就绪（不回零）
```

条件：上次已完成过归零 + 位置有效 + 累计移动不超过 500 次。

### 强制回零

累计移动 500 次后，上电会自动回零一次，确保位置精度。

## 基本操作

### 归零

发送 HOMING 命令，等待 HOMING_DONE 响应。

```
发送: A5 01 00 00 73 28
等待: A5 01 00 0F xx xx    ← 归零完成
```

### 设置倍率

参数 = 目标倍率 × 10。例如 2.0× = 20。

```
发送: A5 10 00 14 xx xx    ← SET_ZOOM(20)
等待: A5 02 00 0A xx xx    ← ARRIVED（到位）
```

支持的倍率：6(0.6×) 10(1.0×) 15 20 25 30 35 40 45 50 55 60 65 70(7.0×)

### 倍率递增/递减

```
递增: A5 11 00 01 xx xx    ← ZOOM_INC(1)
递减: A5 12 00 01 xx xx    ← ZOOM_DEC(1)
```

### 查询状态

```
查询倍率: A5 20 00 00 xx xx
查询状态: A5 21 00 00 xx xx
查询速度: A5 22 00 00 xx xx
查询类型: A5 23 00 00 xx xx
查询范围: A5 24 00 00 xx xx
查询版本: A5 25 00 00 xx xx
```

### 调速

速度值范围 0~1000（占空比 × 10，如 300 = 30%）。

```
设置速度:     A5 60 01 2C xx xx    ← SET_SPEED(300)
设置最小速度: A5 63 00 28 xx xx    ← SET_MIN_SPEED(40)
设置最大速度: A5 64 01 2C xx xx    ← SET_MAX_SPEED(300)
速度+1:       A5 61 00 00 xx xx
速度-1:       A5 62 00 00 xx xx
```

速度参数保存在 Flash，断电不丢失。

### 紧急停止

```
发送: A5 02 00 00 xx xx    ← FORCE_STOP
```

电机立即停止。

## CRC 计算

帧中 `xx xx` 表示 CRC16-Modbus 校验值（小端序），计算范围为帧的前 4 字节。

Python 计算示例：

```python
def crc16(data):
    crc = 0xFFFF
    for b in data:
        crc ^= b
        for _ in range(8):
            crc = (crc >> 1) ^ 0xA001 if crc & 1 else crc >> 1
    return crc & 0xFFFF

# 示例: HOMING 命令
data = bytes([0xA5, 0x01, 0x00, 0x00])
crc = crc16(data)  # 0x2873
frame = data + bytes([crc & 0xFF, crc >> 8])  # A5 01 00 00 73 28
```

## 故障排除

### 上电后电机不动

1. 检查供电电压是否 > 8.3V
2. 检查 UART 接线（TX/RX 是否交叉连接）
3. 发送 QUERY_STATUS 确认设备是否响应
4. 如无响应，断电重启

### 发命令无响应

1. 确认波特率 115200, 8N1
2. 确认 CRC 计算正确
3. 确认帧头为 0xA5
4. 确认 TX/RX 没有接反

### 电机堵转停止 (收到 0xE1)

1. 检查镜头机械是否卡阻
2. 发送 HOMING(0x01) 重新归零
3. 如反复堵转，检查编码器接线

### 过流保护 (收到 0xE2)

1. 立即检查机械卡阻
2. 检查电机接线是否短路
3. 排除后发送 HOMING(0x01) 恢复

### 设置倍率后电机抖动

1. PID 参数可能不匹配当前负载
2. 联系开发人员调整 PID 参数
