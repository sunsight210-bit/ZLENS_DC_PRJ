# ZLENS_DC 通信协议规格书 v2.7

## 物理层

| 参数 | 值 |
|------|-----|
| 接口 | UART (TTL 3.3V) |
| 波特率 | 115200 |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验 | 无 |
| 流控 | 无 |

接线：TX、RX、GND 三根线。

## 帧格式

### 工作模式（6 字节）

```
┌────────┬─────┬─────────┬─────────┬────────┬────────┐
│ Header │ CMD │ Param_H │ Param_L │ CRC_L  │ CRC_H  │
│  0xA5  │     │         │         │        │        │
│  1B    │ 1B  │   1B    │   1B    │   1B   │   1B   │
└────────┴─────┴─────────┴─────────┴────────┴────────┘
```

- Header: 固定 0xA5
- Param: 大端序 (Param_H << 8 | Param_L)
- CRC: CRC16-Modbus，计算范围为前 4 字节，小端序存放

### 工厂模式（8 字节）

```
┌────────┬─────┬──────────┬──────────┬──────────┬──────────┬────────┬────────┐
│ Header │ CMD │ ParamH_H │ ParamH_L │ ParamL_H │ ParamL_L │ CRC_L  │ CRC_H  │
│  0xA5  │     │          │          │          │          │        │        │
│  1B    │ 1B  │    1B    │    1B    │    1B    │    1B    │   1B   │   1B   │
└────────┴─────┴──────────┴──────────┴──────────┴──────────┴────────┴────────┘
```

CRC 计算范围为前 6 字节。

### 握手回显

收到有效帧后，设备先回显原始帧（echo），再发送响应帧。

## CRC16-Modbus 算法

```python
def crc16(data: bytes) -> int:
    crc = 0xFFFF
    for b in data:
        crc ^= b
        for _ in range(8):
            if crc & 1:
                crc = (crc >> 1) ^ 0xA001
            else:
                crc >>= 1
    return crc & 0xFFFF
```

示例：`A5 01 00 00` → CRC = `0x2873` → 帧 = `A5 01 00 00 73 28`

## 命令表

### 控制命令 (0x0X)

| CMD | 名称 | 参数 | 响应 CMD | 响应参数 | 说明 |
|-----|------|------|----------|----------|------|
| 0x01 | HOMING | 0 | 0x01 | 0x000F | 归零校准 |
| 0x02 | FORCE_STOP | 0 | — | — | 紧急停止 |
| 0x03 | GET_STALL_CNT | 0 | 0xE3 | count | 查询堵转次数 |

### 变倍命令 (0x1X)

| CMD | 名称 | 参数 | 响应 CMD | 响应参数 | 说明 |
|-----|------|------|----------|----------|------|
| 0x10 | SET_ZOOM | zoom×10 | 0x10 + 0x02 | zoom×10 + 0x000A | 设置倍率，到位后发 ARRIVED |
| 0x11 | ZOOM_INC | step | 0x10 + 0x02 | zoom×10 + 0x000A | 倍率递增 |
| 0x12 | ZOOM_DEC | step | 0x10 + 0x02 | zoom×10 + 0x000A | 倍率递减 |

示例：设置 2.0× → 参数 = 20 → `A5 10 00 14 CRC_L CRC_H`

## 错误与异常响应

| 响应 CMD | 名称 | 参数 | 触发条件 |
|----------|------|------|----------|
| 0x03 | REQ_INVALID | 0x000E | 系统忙时收到运动命令 |
| 0xE0 | ERR_PARAM | — | 参数超范围 |
| 0xE1 | STALL_STOP | — | 电机堵转停止（电流超阈值 1000ms） |
| 0xE2 | OVERCURRENT | — | 过流保护触发（电流超阈值 50ms） |

### 异常恢复

| 异常 | 恢复方法 |
|------|----------|
| 堵转停止 (0xE1) | 发送 HOMING(0x01) 重新归零 |
| 过流 (0xE2) | 检查机械卡阻，发送 HOMING 重新归零 |
| 无响应 | 检查接线和供电，断电重启 |

## 通信示例

### 归零

```
TX: A5 01 00 00 73 28    → HOMING
RX: A5 01 00 00 73 28    → 握手回显
RX: A5 01 00 0F xx xx    → HOMING_DONE (到位后)
```

### 设置 2.0× 倍率

```
TX: A5 10 00 14 xx xx    → SET_ZOOM(20)
RX: A5 10 00 14 xx xx    → 握手回显
RX: A5 10 00 14 xx xx    → ZOOM 响应
RX: A5 02 00 0A xx xx    → ARRIVED (到位后)
```

### 查询当前倍率

```
TX: A5 20 00 00 xx xx    → QUERY_ZOOM
RX: A5 20 00 00 xx xx    → 握手回显
RX: A5 10 00 14 xx xx    → ZOOM = 20 (2.0×)
```
